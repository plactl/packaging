# plactl

**plactl** is an internal developer platform: GitHub-integrated, Terraform-driven
AWS provisioning that runs entirely in your own AWS Organization.

This crate is the **official name reservation** by the plactl project
([github.com/plactl](https://github.com/plactl)). The plactl CLI has not shipped
to crates.io yet; when it does, releases will be published here.

Installing this version gives you a stub binary that prints project information
and exits.

If you believe this crate conflicts with your project, contact the author via
the GitHub organization above.
